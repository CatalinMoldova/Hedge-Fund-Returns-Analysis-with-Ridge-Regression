## Hedge Fund Returns Analysis with Ridge Regression

A comprehensive statistical analysis project implementing ridge regression, bias-variance tradeoff analysis, and feature engineering on hedge fund datasets using Fama-French five-factor models.

### Project Overview

This project explores advanced regression techniques and their applications to financial data analysis. It includes theoretical derivations of ridge regression properties, loss function optimization, and practical implementation of feature engineering on hedge fund return data.

### Key Components

**Bias-Variance Tradeoff Analysis**: Mathematical derivation of ridge regression estimator bias and variance, including optimal regularization parameter calculation using MSE minimization.

**Beyond Quadratic Loss**: Exploration of L1 loss functions and their applications to financial forecasting, with geometric median interpretation for multi-dimensional cases.

**Feature Engineering on Financial Data**: Implementation of regression models using Fama-French five-factor data (market risk premium, SMB, HML, RMW, CMA) merged with hedge fund returns to analyze performance drivers.

### Technologies Used

Python with pandas, NumPy, and scikit-learn for data processing and statistical modeling. Google Colab for development and Google Drive integration for data management.

### Dataset

Fama-French Research Data (5 Factors) covering market returns from 1963 onwards, combined with hedge fund performance metrics including excess returns and factor exposures.

### Implementation Highlights

Ridge regression with regularization parameter optimization, statistical analysis of fund performance metrics (mean, standard deviation, skewness, kurtosis), and time-series feature engineering for financial forecasting.
